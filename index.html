Portfólio de Ciência de Dados

O objetivo deste repositório é apresentar alguns projetos de ciência de dados, para fins acadêmicos, autoaprendizagem e profissionais. Aqui você encontra soluções para projetos na área de Data Science, Machine Learning e Data Analysis. Todo o material pode ser consultado através de um “README” e arquivos com códigos em python.

Meus Projetos:

1. Projetos com Machine Learning:

  • Insurance All Company [ENG] - O modelo foi criado para calcular a taxa churn de uma empresa de seguros de vida, ele calcula o tamanho do time de marketing para atingir um aumento de 80% da clientela.
  • Blocker Fraud Company [ENG] - Proposta de modelo de Machine Learning para solucionar um problema de classificação com o objetivo de rastrear transações fraudulentas, o modelo apresentou uma acurácia de 99% e uma taxa de ganho de aproximadamente 300%.
  • OkCupid-Date-A-Scientist [ENG] - Através do aplicativo de relacionamento OkCupid prevemos o status de relacionamento das pessoas, através dessa predição pode-se melhorar o marketing para dia dos namorados e afins. O modelo apresenta acurácia de 98%.
  • Spotify Songs Popularity Prediction [ENG] - Através das características técnicas das músicas encontradas no Spotify, prevemos o seu sucesso utilizando métricas criadas pelo próprio aplicativo para calculo do grau de popularidade de uma música.

2. Data Visualization:

  • Life Expectancy and GDP - Análise do PIB e expectativa de vida entre 6 países.
  • Biodiversity [ENG] - Análise dos dados para investigar os status de conservação de espécies no National Park Service.

3. Análise com syntax em Python:

  • U.S Medical Insurance Costs- Um projeto de análise de dados sobre custos de seguro médico nos EUA utilizando syntax do Python.
